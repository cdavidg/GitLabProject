# GitLabProject

This lab demonstrates Git and GitHub workflow using Visual Studio Code.

## Project Overview

This project is part of a Git and GitHub learning exercise that covers:

- Creating and managing repositories
- Working with branches
- Making commits and push changes
- Creating pull requests
- Collaborative development workflow

## Files Included

- `info.txt` - Personal information and favorite programming language
- `goals.txt` - Career goals and future plans

## Technologies Used

- Git & GitHub
- Visual Studio Code
- Markdown

## Learning Objectives

- Understanding version control with Git
- Practicing GitHub workflow
- Working with branches and pull requests
- Managing code collaboration
